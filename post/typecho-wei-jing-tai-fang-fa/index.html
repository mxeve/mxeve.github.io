Typecho伪静态方法

2020-10-04 / 1 min read

复制下方Typecho Apache伪静态伪静态规则，保存并命名为.htaccess，将.htaccess上传到网站根目录即可（宝塔直接添加以下规则）

RewriteEngine On RewriteBase / RewriteRule ^index\.php$ - [L] RewriteCond %{REQUEST_FILENAME} !-f RewriteCond %{REQUEST_FILENAME} !-d RewriteRule . /index.php [L]

复制下方Typecho Nginx伪静态规则，保存为typecho.conf，在Nginx的网站配置文件中调用typecho.conf伪静态规则：（宝塔直接添加以下规则）

if (-f $request_filename/index.html){
rewrite (.) $1/index.html break;
}
if (-f $request_filename/index.php){
rewrite (.) $1/index.php;
}
if (!-e $request_filename){
rewrite (.*) /index.php;
}

然后在Typecho控制台选择“设置”--“永久链接”中，启用重写功能，在自定义文章路径中个性化定义写入：/{cid}.html
